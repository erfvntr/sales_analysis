Overview:

This project focuses on analyzing sales data using Python, SQL and Tableau. The project includes data preprocessing, exploratory data analysis (EDA), visualization of important metrics, statistical tests, analysis using BI. 


 Technologies Used in Python:
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib 
- Seaborn
- Scipy


The notebook includes the following stages:

1. Data Loading
- Importing the dataset from Google BigQuery using SQL-request

2. Data Cleaning
- Checking missing values, handling duplicates and outliers

3. Exploratory Data Analysis (EDA)
- Statistical overview of the dataset and аnalysis of sales indicators, іdentification of trends and patterns

4. Data Visualization
- line charts
- Bar charts
- Histograms
- Pie charts
- Heatmap
- Scatter plots

5. Creating Pivot Tables

6. Statistical correlation analysis:
- Normal test (normaltest, kendalltau, spearmanr)
- Pearson Correlation

7. Statistical analysis of group differences
- proportions_ztest
- f_oneway
- kruskal
